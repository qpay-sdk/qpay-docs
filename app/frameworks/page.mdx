# Framework Packages

QPay SDK-уудын дээр суурилсан framework-specific integration package-ууд. Тухайн framework-ийн convention дагаж, DI, config, webhook, UI component-уудыг бэлнээр хангана.

| Framework | Package | Registry | GitHub |
|-----------|---------|----------|--------|
| Laravel | `qpay-sdk/laravel` | Packagist | [qpay-laravel](https://github.com/qpay-sdk/qpay-laravel) |
| Django | `django-qpay` | PyPI | [django-qpay](https://github.com/qpay-sdk/django-qpay) |
| Spring Boot | `qpay-spring-boot-starter` | Maven Central | [qpay-spring-boot-starter](https://github.com/qpay-sdk/qpay-spring-boot-starter) |
| Express | `@qpay-sdk/express` | npm | [qpay-express](https://github.com/qpay-sdk/qpay-express) |
| NestJS | `@qpay-sdk/nestjs` | npm | [qpay-nestjs](https://github.com/qpay-sdk/qpay-nestjs) |
| Rails | `qpay-rails` | RubyGems | [qpay-rails](https://github.com/qpay-sdk/qpay-rails) |
| ASP.NET Core | `QPay.AspNetCore` | NuGet | [qpay-aspnetcore](https://github.com/qpay-sdk/qpay-aspnetcore) |
| Flutter | `qpay_flutter` | pub.dev | [qpay-flutter](https://github.com/qpay-sdk/qpay-flutter) |
| FastAPI | `fastapi-qpay` | PyPI | [fastapi-qpay](https://github.com/qpay-sdk/fastapi-qpay) |
