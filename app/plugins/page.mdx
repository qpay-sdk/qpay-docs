# CMS Plugins

QPay V2 төлбөрийн integration-ууд e-commerce болон CMS платформуудад. Plugin-ууд QPay V2 API-тай шууд ажилладаг (SDK шаардлагагүй).

| Platform | Plugin | GitHub |
|----------|--------|--------|
| WooCommerce | `qpay-woocommerce` | [qpay-woocommerce](https://github.com/qpay-sdk/qpay-woocommerce) |
| Shopify | `qpay-shopify` | [qpay-shopify](https://github.com/qpay-sdk/qpay-shopify) |
| OpenCart | `qpay-opencart` | [qpay-opencart](https://github.com/qpay-sdk/qpay-opencart) |
| Magento | `qpay-magento` | [qpay-magento](https://github.com/qpay-sdk/qpay-magento) |
| PrestaShop | `qpay-prestashop` | [qpay-prestashop](https://github.com/qpay-sdk/qpay-prestashop) |
| WordPress | `qpay-wordpress` | [qpay-wordpress](https://github.com/qpay-sdk/qpay-wordpress) |
| Odoo | `qpay-odoo` | [qpay-odoo](https://github.com/qpay-sdk/qpay-odoo) |
